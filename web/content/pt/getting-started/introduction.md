# Introducao

Ja desejou que seu assistente de IA tivesse colegas de trabalho? E exatamente isso que o oh-my-agent faz.

Em vez de uma unica IA fazendo tudo (e se confundindo no meio do caminho), o oh-my-agent distribui o trabalho entre **agentes especializados** -- um especialista em frontend, um em backend, um revisor de QA, um planejador de PM e mais. Cada agente conhece profundamente seu dominio e tem suas proprias ferramentas, checklists e playbooks de erros.

## O Que Voce Realmente Recebe

- **14 agentes especializados** -- cada um modelado a partir de um papel real em uma equipe de engenharia
- **Comandos slash** como `/plan`, `/coordinate`, `/review` que ativam workflows estruturados
- **Deteccao automatica** -- apenas descreva o que voce quer em linguagem natural e o workflow certo e ativado (em 11 idiomas!)
- **Execucao paralela** -- inicie multiplos agentes trabalhando em diferentes partes simultaneamente
- **Dashboards em tempo real** -- terminal e web UI para monitorar o que seus agentes estao fazendo
- **Funciona em qualquer lugar** -- Antigravity, Claude Code, Cursor, Gemini CLI, Codex CLI, OpenCode e mais

## Conheca Sua Equipe

| Agente | O Que Faz |
|--------|----------|
| **oma-brainstorm** | Explora ideias antes de voce se comprometer a construir qualquer coisa |
| **oma-pm** | Decompoe requisitos, planeja tarefas, define contratos de API |
| **oma-frontend** | Constroi UI com React/Next.js, TypeScript e Tailwind CSS |
| **oma-backend** | Cria APIs em Python, Node.js ou Rust |
| **oma-db** | Projeta schemas de banco de dados, gerencia migracoes, otimiza queries |
| **oma-mobile** | Constroi apps cross-platform com Flutter |
| **oma-design** | Cria design systems com tokens, acessibilidade e regras responsivas |
| **oma-qa** | Revisa seguranca (OWASP), performance e acessibilidade |
| **oma-debug** | Encontra causas raiz, escreve correcoes e testes de regressao |
| **oma-tf-infra** | Provisiona infraestrutura cloud com Terraform |
| **oma-dev-workflow** | Automatiza CI/CD, releases e tarefas de monorepo |
| **oma-translator** | Traduz conteudo naturalmente entre idiomas |
| **oma-orchestrator** | Executa multiplos agentes em paralelo via CLI |
| **oma-commit** | Cria commits convencionais limpos |

## Como Funciona (A Versao de 30 Segundos)

1. Voce descreve o que quer construir
2. O oh-my-agent descobre quais agentes sao necessarios
3. Os agentes executam com seu conhecimento especializado
4. Portoes de qualidade verificam o trabalho antes de concluir
5. Voce recebe codigo pronto para producao, nao apenas sugestoes

## A Ideia Central

Tudo vive em `.agents/` -- um unico diretorio portatil no seu projeto. Skills, workflows, configs, tudo. Troque de IDE quando quiser. Sua configuracao de agentes viaja com seu codigo.
